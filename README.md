# MistySoft QR-NFC Review

Landing page for QR-NFC SaaS service.

## Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Alpine.js (CDN)

## Setup

1. Open `index.html` in browser
2. Or deploy to hosting (InfinityFree, GitHub Pages, etc.)

## Deployment

Deploy to InfinityFree:
1. Upload `index.html` to `htdocs/`
2. Configure subdomain: review.mistydev.id.vn
